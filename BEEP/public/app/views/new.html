v2.2.1 (15-07-2020) - Current version

  • Hive change log: every hive configuration change, or relocation is filed as a new inspection, so you can keep track of your historic changes
  • Inspection sample code: added (lab) sample code generator as an inspection item. It generates a unique sample code that can be put on a sample for the lab to register research results
  • Inspection Liebefeld: Added honey super input fields to Liebefeld colony size calculation
  • Data export: ability to also export data of collaboration group hives with measurement systems
  • Data export: bugfix that prevented from exporting CSV data files from selected dates and measurement systems
  • BEEP base: support for new iOS and Android app output to configure your BEEP base with the native apps
  • Bugfixes: small fixes for issues received as feedback

v2.2.0 (25-02-2020)

  • Create new Apiary wizard, leading you through all the steps in ordered sequence
  • New fields in Apiaries and Hives e.g. roofed, hive layer/frame dimensions, queen notes, etc.
  • Research: ability to consent to sharing your data with research
  • Helpdesk: search the knowledge base for help, or file a support ticket directly from the app
  • Weather: See apiary weather info (if lat/lon coordinates have been filled and you have installed a BEEP base) powered by Darksky API
  • Export: export and download sensor measurements to a CSV file if you have installed a BEEP base
  • {{lang.Whats_new}}: this in-app overview of new features with links to the features themselves
  • Bugfixes: small fixes for issues received as feedback
